# atlas.windowcontroller

![Banner](banner-image.png)

Cross-platform window manager and WiFi controller. Control your desktop windows from any device on the same WiFi network.

## Features
- List all open windows.
- Focus any window.
- Snap windows to positions (Top Left, Top Right, Bottom Left, Bottom Right, Left, Right, Center, Fullscreen).
- Launch applications remotely.
- Mobile-friendly Web UI.

## Usage
1. Run the application.
2. Note the IP address and port (default 5000) displayed in the terminal.
3. Open the URL on your mobile device or another PC on the same WiFi.

## Build
Uses `gobake` for builds.
```powershell
gobake build
```
